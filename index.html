Install

One command on every platform. Prebuilt, checksum-verified binaries.

Recommended

macOS — Homebrew

Taps the formula, verifies the checksum, installs to your PATH.

brew install datamaia/tap/andromeda

Linux / macOS — script

Detects your OS/arch and downloads the matching release binary.

curl -fsSL https://andromedacli.com/install | bash

Windows — PowerShell

SHA256-verified; installs andromeda.exe and adds it to your PATH.

irm https://andromedacli.com/install.ps1 | iex

With Go

Compile and install from source with the Go toolchain.

go install github.com/datamaia/andromeda/cmd/andromeda@latest

Why Andromeda

A coding agent that respects your machine, your model choice, and your permissions.

🔒

Local-first & private

No cloud, no telemetry. Credentials live in your OS keychain.

🔌

Vendor-agnostic

Anthropic, OpenAI (API or ChatGPT), Gemini, xAI, Groq, Ollama, and more — switch with a flag.

🛡️

Safe by default

Every write, command, and network call is read-only until you grant it — or allow-list it.

⌨️

CLI & TUI

Script one-shot runs, or drop into a full-screen terminal UI with slash commands and a plan/approve flow.

💾

Persistent sessions

Conversations are remembered across turns and resume with --continue.

📄

AGENTS.md-aware

Project guidance is folded into the agent's context on every run. Scaffold it with /init.
